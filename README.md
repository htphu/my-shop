
# Chức năng cơ bản

Đây là một Website bán hàng có những chức năng cơ bản sau:
- Đăng nhập, đăng xuất, đăng ký
- Filter, search
- Đặt hàng, bình luận
- Quản lý sản phẩm, thể loại, người dùng, đặt hàng (CRUD)




## Công nghệ sử dụng

**FrontEnd:** React, Material UI, Axios

**BackEnd:** Node, ExpressJS, MySQL, JWT
